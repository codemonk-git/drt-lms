## Quick Reference

### Creating a Lead
```bash
1. Add record to: database/leads/leads.json
2. Log activity to: database/audit/lead_activities.json
3. Create assignment: database/assignments/assignments.json
4. Set SLA: database/audit/sla_tracking.json
```

### Accessing Data by Category
```bash
# Organization data
ls database/tenants/

# Lead information
ls database/leads/

# Current assignments
ls database/assignments/

# Audit trail & compliance
ls database/audit/

# Business rules
ls database/rules/

# Help & documentation
ls database/docs/
```
